About This Setup

This website is hosted on Amazon S3 (Simple Storage Service), which provides a simple, scalable solution for static website hosting. S3 offers high availability, durability, and performance at a low cost.

Combined with Amazon CloudFront (CDN), Route 53 (DNS), and other AWS services, you can build a highly available and performant static website infrastructure that scales automatically to meet demand.
